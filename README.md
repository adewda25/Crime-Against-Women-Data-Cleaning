# Crime Against Women Data Cleaning Project

## Project Overview

This project focuses on cleaning and preprocessing Crime Against Women datasets using Python and Pandas. The objective is to transform raw data into a clean, structured format suitable for analysis, visualization, and dashboard development.

## Objectives

* Clean and preprocess multiple crime datasets
* Handle missing and inconsistent values
* Standardize column names and data formats
* Prepare datasets for analytics and reporting
* Create a reliable dataset for Power BI dashboard development

## Dataset Information

The project includes multiple datasets related to crimes against women across different states and years.

Key attributes include:

* State
* Year
* Cases Reported
* Cases Chargesheeted
* Cases Convicted
* Cases Acquitted
* Trials Completed
* Pending Investigation
* Pending Trial

## Data Cleaning Process

The following steps were performed:

1. Data Inspection

   * Checked dataset structure
   * Identified missing values
   * Examined data types

2. Data Cleaning

   * Removed duplicates
   * Handled missing values
   * Corrected inconsistent column names
   * Standardized formatting

3. Data Preparation

   * Converted data types
   * Prepared datasets for merging
   * Validated cleaned data

4. Final Output

   * Generated cleaned datasets ready for analysis and visualization
